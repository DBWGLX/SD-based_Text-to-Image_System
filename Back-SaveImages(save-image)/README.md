# **/图像和历史记录保存**

*Contributed By:*  Yilin Xu <br>

**前端返回**：一个图像数据（userId，imageUrl）封装在图像类中。<br>

**测试配置**：将配置文件application.yml中的数据库连接自己的。<br>

**测试：**
地址：localhost:8080/api/images/save <br>
需要自己先在user表中手动添加一个可以用的数据，再进行测试
